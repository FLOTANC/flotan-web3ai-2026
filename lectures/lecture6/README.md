## 🤖 v2 の目玉機能：Wake先生（AIアシスタント）

**Wake先生（ウェイク先生）**

> 子どもの表現の"航跡"を見つけ、次の一歩を照らす先生。  
> 「正解を教える先生」ではなく、*子どもの航跡を一緒に見つける優しいAI先生* です。

### 機能
- 幼稚園〜中学生が美術・アート・画材・色彩・技法について質問できる
- Google Gemini 2.5 Flash API で日本語回答（子どもにわかりやすい言葉）
- 答えを直接渡すより「なぜそう思うの？」と考えを引き出す問いかけ重視
- 生徒の名前を呼んで個別に対応
- 会話履歴を Supabase（PostgreSQL）に保存
- 生徒が質問すると Firebase Realtime Database 経由で先生に通知
- 先生は生徒ごとの会話履歴を一覧で確認できる

### 技術構成
| 役割 | 技術 |
|---|---|
| AI | Google Gemini 2.5 Flash（`/api/chat` Vercel Function） |
| 会話履歴DB | Supabase（PostgreSQL）|
| リアルタイム通知 | Firebase Realtime Database |
| バックエンド | Vercel Serverless Functions（`api/chat.js`, `api/history.js`） |

## 🔧 v2 で新たに追加・改善した点

- **Wake先生タブ**を新設（AI先生への質問、会話履歴の保存と閲覧）
