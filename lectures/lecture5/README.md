## 🔄 PIVOTの経緯

VPC v1では、家庭内の課題である「夫の帰宅通知・お迎えリマインド・提出物管理」を解決するために、Family Compassを制作しました。実際に使ってみた結果、日々の予定共有やリマインドの仕組みとして一定の手応えがあり、満足のいくプロトタイプになりました。

その経験を踏まえ、次に取り組んだのが、子どもの絵にフィードバックを届けるためのプロトタイプです。

きっかけは、以前ミニバスのママ友から「息子に絵を教えてほしい」と頼まれていたことでした。また、他の幼稚園のママたちからも「子どもの絵を見てほしい」と相談されることがあり、身近なところに同じようなニーズがあると感じました。

そこで、子どもの作品を共有し、それに対してやさしく具体的なコメントを返せるようなサービスのプロトタイプを制作しました。

---

## 🎯 最小スコープ

「先生が生徒の作品にコメントを返せる」「保護者が作品を見られる」この2点が動けば価値が伝わると判断し、以下に絞りました。

- 作品のアップロード・一覧表示（3Dギャラリー）
- 先生による講評コメントの投稿
- 保護者・生徒への連絡機能
- 授業情報（次の授業・モチーフ・Zoomリンク）の共有
- 成長記録の蓄積

## 💬 Discord 投稿文

**🔄 PIVOT あり｜絵画教室の生徒・先生・保護者をつなぐアプリを作りました**

**【バグ】**  
絵画教室で描いた作品に、プロの目線でフィードバックをもらえる機会がない。先生とのやり取りはLINEに散らばり、作品も残らない。「どこが良かったか」が言語化されないまま、子どもは次の絵に進んでしまう。

**【誰のため】**  
絵画教室の先生・生徒（幼稚園〜中学生）・保護者

**【何が動くか】**  
作品を3Dギャラリーに展示 → 先生が講評コメントを投稿 → 保護者も作品を閲覧・コメント。連絡機能・授業情報・成長記録もまとめて管理できます。  
🔗 https://art-feedback.vercel.app

**【PIVOTについて】**  
VPC v1では「夫の帰宅通知・お迎えリマインド・提出物管理」を解くFamily Compassを作りました。
https://family-compass-seven.vercel.app
一定の手応えを感じながらも、ミニバスのママ友や幼稚園のママたちから「子どもの絵を見てほしい」という声が身近にあることに気づき、PIVOTしました。
